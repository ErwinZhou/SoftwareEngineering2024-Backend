# Software-Engineering-backend
The backend of our Software-Engineering homework

content tree:

```
/Software-Engineering-backend
|-- app/                        # 应用主模块       
|   |-- main.py                 # 应用的入口点，包括路由和中间件
|   |-- settings.py             # 配置文件，如环境变量
|   |-- api/                    # API 特定文件
|   |-- models/                 # 数据模型目录
|   |-- schemas/                # Pydantic 模型（用于请求和响应）
|   |-- services/               # 业务逻辑层 CRUD 操作
|-- tests/                      # 测试目录
|-- requirements.txt            # 项目依赖文件
```
